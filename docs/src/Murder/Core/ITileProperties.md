**Namespace:** Murder.Core \
**Assembly:** Murder.dll


Interface for classes that have tile properties.

**Intent:** Marks a type as a carrier of per-tile property data (e.g. floor type, surface friction) for use by map and rendering systems.

**Use-case:** Implement this interface on component or asset types that need to associate additional metadata with individual map tiles.
