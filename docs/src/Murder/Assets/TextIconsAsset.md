# TextIconsAsset

**Namespace:** Murder.Assets \
**Assembly:** Murder.dll

```csharp
public class TextIconsAsset : GameAsset
```

Collects all the small icons that can be embedded inline within rendered text (dialogue, story text, etc.), each identified by a string ID.

**Intent:** Give writers a single, editor-managed catalog of inline icons that dialogue/story text can reference by name, so the text rendering pipeline can substitute a placeholder in a string for the correct `Portrait` image without any code changes per icon.

**Use-case:** Create (or use the project's existing) `TextIconsAsset`, add entries to `Icons` in the editor (each an `Id` plus a `Portrait`), and reference an icon from dialogue text using its ID. At draw time the text renderer calls `TryFetchIcon(id)` to resolve the icon and draw it inline with the surrounding glyphs.
