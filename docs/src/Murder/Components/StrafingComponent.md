# StrafingComponent

**Namespace:** Murder.Components \
**Assembly:** Murder.dll

```csharp
public sealed struct StrafingComponent : IComponent
```

Marker component that enables lateral (strafing) movement for an agent, allowing it to move sideways while keeping its facing direction.

**Intent:** Opt an agent into strafing movement behaviour.

**Use-case:** Attach to enemy or player entities that should be able to side-step without turning, such as characters that always face a target while moving around it.
