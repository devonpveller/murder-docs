# ICommands

**Namespace:** Murder.Diagnostics \
**Assembly:** Murder.dll

```csharp
public abstract ICommands
```

Implemented by static classes that would like to export command functionality.

**Intent:** Marks a static class as a provider of in-game console commands.

**Use-case:** Implement this interface on a static class and annotate its methods to make them callable from the in-game debug console.
