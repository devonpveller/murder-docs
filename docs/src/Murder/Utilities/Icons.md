# Icons

**Namespace:** Murder.Utilities \
**Assembly:** Murder.dll

```csharp
public static class Icons
```

Named `char` constants for editor icon glyphs drawn from the embedded icon font.

**Intent:** Centralizes all editor icon character codes so that editor rendering code can reference icons by name rather than raw Unicode values.

**Use-case:** Pass these constants to any editor UI rendering call that draws text icons for entities, components, and world objects.
